== Description ==

Appends .htm or .html or .php or oter extension to the URL of PAGES when using seo permalinks. Go to plugin setting page from tools menu to add whatever extension you like!
<br />
For support or comment please go <a href="http://wpintent.com/wordpress-plugins/wordpress-pages-with-extension/">Wordpress Page Extension Plugin</a>

== Installation ==

1. Upload pages-with-extension.php to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
